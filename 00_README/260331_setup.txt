5. GIT 설치 
 1) https://git-scm.com/install/windows
 2) 브라우저에서 다운로드 후 설치 -> PATH 등록 설정
 3) 아래 코드를 프로젝트 폴더에서 실행(파워쉘)
  - git init							                    # git 설정 초기화
  - git config --global user.name "mercy"	    # 사용자 정보 설정
  - git config --global user.email "leehyukjun1@gmail.com"
  - git add .							                    # 현재 위치 모두 스테이징 
  - git commit -m "first commit"				      # 로컬 저장
  4) 파일 수정하여 git 서버에 등록하는 순서
  - 1. 파일 수정 (Working Directory)
  - 2. git add (staging)
  - 3. git commit (로컬 저장)
  - 4. git push (서버 업로드)
  - 명령어
   # git status     # 현재 상태 확인
   # git remote -v  # 원격 저장소 확인
   # git branch     # 브랜치 확인
   # git add .
   # git commit -m "update"
   # git push -u origin main
