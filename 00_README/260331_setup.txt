
0. 구조
 [Windows] -> Windows는 IDE와 Docker Desktop만 설치
   └── VSCode (UI)
   └── Docker Desktop (WSL2 Backend)
   └── WSL Ubuntu (Linux VM)
 [WSL Ubuntu] ← 실제 개발 환경
   ├── git      
   ├── python
   ├── Source Code      
   └── venv

1. WSL 설치
 1) 윈도우 11 파워쉘
   - # wsl --update
   - # wsl --install -d Ubuntu

2. UBUNTU 설치
 1) root 계정 설정
  - # sudo -i
  - # sudo passwd root
  - # su -
 2) UPDATE
  - # sudo apt update && sudo apt upgrade -y
  - # sudo apt install git python3 python3-pip python3-venv -y
  - # sudo apt install build-essential curl wget -y
 3) wsl Ubuntu에서 삭제 및 확인
  - # 삭제 -> # wsl --unregister Ubuntu
  - # 확인 -> # wsl -l -v

3. VSCODE 설치
 1) https://code.visualstudio.com 다운로드 후 설치
 2) Theme(File → Preferences → Color Theme)
 3) Extension Pack
  (1) WSL Extension(MS) -> 설치후 Connect to WSL
  (2) Clone Repository -> Git 선택 후 -> 5번 최초 설치 진행
  (3) 아래는 선택
  - Language : Python, Pylance
  - DevOps : Docker, Dev Containers, Remote-SSH
  - API    : REST Client, GitLens
  - Log    : PostGre, Log Viewer
  - AI     : Windsurf, Github Copilot, ChatGPT
  5) 삭제 방법
   - Program 삭제
   - 설정삭제 ->C:\Users\<사용자명>\AppData\Roaming\Code
   - 설정삭제 ->C:\Users\<사용자명>\.vscode
   - 캐시삭제 ->C:\Users\<사용자명>\.gitconfig

5. GIT 최초 설치 및 PUSH/PULL
 1) git 설치  
  - # sudo apt install git -y
 2) 환경설정(개발 폴더에서 진행)
  - git init							                    # git 설정 초기화(최초1회)
  - git config --global user.name "mercy"	    # 사용자 정보 설정
  - git config --global user.email "leehyukjun1@gmail.com"
  - git add .							                    # 현재 위치 모두 스테이징 
  - git commit -m "first commit"				      # 로컬 저장
 3) 파일 수정하여 git 서버에 등록하는 순서
  - 1. 파일 수정 (Working Directory)
  - 2. git add (staging)
  - 3. git commit (로컬 저장)
  - 4. git push (서버 업로드)
  - 명령어
   # git status     # 현재 상태 확인
   # git remote -v  # 원격 저장소 확인
   # git branch     # 브랜치 확인
   # git add .
   # git commit -m "update"
   # git push -u origin main
   # 싱크 안 맞을 경우 -> git pull --rebase origin main 후 위 push 실행
